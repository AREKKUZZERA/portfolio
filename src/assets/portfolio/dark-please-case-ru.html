01 / задача

Темный режим должен ощущаться безопасным

Главный риск расширения — пользователь не понимает, меняет ли он один сайт или весь браузинг. Поэтому интерфейс разделяет ежедневные быстрые действия и долгосрочные настройки.

Ясная область действия

Глобальное включение и текущий сайт визуально разделены, чтобы пользователь понимал масштаб изменения.

Быстрая настройка

Яркость, контраст и движок темной темы доступны в popup, потому что напрямую влияют на чтение.

Безопасный откат

Исключения, импорт, экспорт и сброс вынесены в настройки, чтобы не перегружать основной сценарий.

02 / структура

Два слоя продукта вместо одного перегруженного popup

Popup решает задачу “что сделать прямо сейчас”, а Options page отвечает на вопрос “как расширение должно вести себя дальше”.

03 / интерфейс

UI работает как тихая системная панель

Визуальный язык продолжает темную тему: плотные панели, розовый акцент, компактные элементы управления и понятные состояния.

Popup

Содержит только ежедневные действия: включение, текущий сайт, интенсивность и переход в настройки.

Options page

Разделяет постоянные правила по вкладкам: поведение, исключения, автоматизация, горячие клавиши и восстановление.

04 / результат

Расширение стало понятным и управляемым

Пользователь получает быстрый контроль без ощущения, что расширение случайно изменит весь интернет.
